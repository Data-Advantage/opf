# OPF 0.2.0 Migration Notes

## Chart Catalog ID Rename

The United Kingdom map chart catalog record was corrected to `united-kingdom`. The removed ID was the misspelled United Kingdom slug, with `kingdom` missing its second `g`.

Update OPF documents that reference the removed misspelled chart type:

```diff
- "type": "<removed misspelled United Kingdom slug>"
+ "type": "united-kingdom"
```

This is a breaking catalog ID change for documents or tools that referenced the misspelled slug directly.
